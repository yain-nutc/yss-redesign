| 功能 | 說明 |

| 🎨 **現代 FinTech 風格** | 深海藍 + 金色漸層配色，高端金融科技美學 |

## 🛠 技術棧

- **HTML5** — 語意化標籤、SEO meta tags
- **Vanilla CSS** — CSS Custom Properties 設計系統
- **Vanilla JavaScript** — 無框架依賴
- **Canvas API** — 粒子動畫
- **Intersection Observer API** — 滾動觸發效果
- **Google Fonts** — Noto Sans TC + Inter
- **Font Awesome 6** — 圖示庫

---

## 🚀 本地開發

直接用瀏覽器開啟 `index.html` 即可，無需安裝任何套件或啟動伺服器。

```bash
# 或使用 VS Code Live Server
# 或使用 Python
python -m http.server 8080
```
